# GrandMA3-Midi-Desk

This is the needed Files for an GrandMA3onPC Console using some Midi Controllers and TouchOSC

Also using the Repository https://github.com/einlichtvogel/grandMA3-Feedback-Chataigne-Module

## Prerequisits
 
- *GrandMA3onPC* https://www.malighting.com/downloads/products/grandma3/
- *Chataigne* https://benjamin.kuperberg.fr/chataigne/en#download
- Akai Midi Mix
- Akai APC Mini Mk2
- Launchpad Mini Mk3
- *TouchOSC* on iPad or other Tablet (optional)

## Install

1. Copy the Content of the Folder /Chataigne Modules into the Chataigne Module Folder on your System (usually /Documents/Chatagine/Modules)

2. Using the Midi Mix Editor load the Midimix.midimix onte the Akai Midi Mix

3. Open MA3.noisette with Chataigne and connect the Controllers in the Module Section

4. Open the MA3.tosc with your TouchOSC Device and Connect to the IP of Chataigne

5. Import the MA Setup into the OSC of GrandMA3

6. Configure the GrandMA3 Feedback Plugin

7. Begin your Show
